Dance Duel

'Dance Duel' is a dance contest organized by A.C.E.S. club, where you can showcase your dancing skills and creativity.

Rules and Guidelines

  • Any dance style can be performed.
  • Submit your audio file on aces@dyp.co.in.
    File format should be .mp4.
  • Only solo entries are allowed.
  • Only one entry from each participant is accepted.
  • Performances should be no longer than 3 minutes.
